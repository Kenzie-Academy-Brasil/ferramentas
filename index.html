Codepen

O CodePen é um ambiente de desenvolvimento web na qual não precisamos instalar e configurar absolutamente nada. Basta fazer o cadastro na plataforma seguindo os passos a seguir

1 - Acesse https://codepen.io/

2 - Clique no botão verde escrito SingUp

3 - Escolha uma das formas de cadastro que a plataforma oferece

4 - Após fazer o cadastro você estará na página inicial do codepen.

5 - Agora vamos criar o nosso primeiro projeto web na plataforma clicando em pen no menu esquerdo.

6 - Sua plataforma de desenvolvimento irá parecer como esta

7 - Perfeito, agora estamos prontos para iniciar uma nova jornada no mundo da programação. Aperte os cintos e bora acelerar ao máximo seus conhecimentos web.

    Linux - Ubuntu

    O VS Code é o nosso editor de códigos. Ele será seu melhor amigo, especialmente se você dedicar um tempo para aprender as funções e atalhos dele!

    1 - Entre no site do Visual Studio Code e baixe a versão .deb

    2 - No terminal vá até o diretório onde o arquivo foi baixado e execute o comando:

    sudo apt install ./<nome_do_arquivo>.deb

    3 - Para mais opções de instalação e configuração, consulte a página oficial da instalação do VSCode no Linux
